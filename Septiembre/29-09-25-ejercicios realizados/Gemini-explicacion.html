Síntesis de Nuestra Conversación sobre HTML, CSS y JavaScript

Este documento resume los conceptos clave que hemos cubierto, desde la estructura básica de una página web hasta la manipulación interactiva con JavaScript.

1. El Ejercicio Práctico: Alternar el Color de un Div

El objetivo era crear una página con un botón que, al ser pulsado, cambiara el color de fondo de un `div`. Esta tarea nos sirvió para explorar la interacción entre los tres pilares de la web.

Resultado Final del Ejercicio

HTML (index.html): La estructura de la página.

<!DOCTYPE html>
<html lang="es">
<head>
    <!-- Metadatos esenciales -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejercicio Día 01</title>
    <!-- Enlace fundamental para conectar con la hoja de estilos CSS -->
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <!-- El botón que dispara la acción con el evento onclick -->
    <button onclick="changeColor()">Cambia color</button>

    <!-- El div al que afectará el cambio. Tiene un 'id' para ser encontrado por JS -->
    <div id="caja-colorida">Este es un div</div>

    <!-- El script se coloca al final para asegurar que el HTML se haya cargado primero -->
    <script src="script.js"></script>
</body>
</html>

CSS (style.css): Las reglas de estilo y apariencia.

/* Estilos aplicados a toda la página a través de la etiqueta body */
body {
    font-family: 'Courier New', Courier, monospace;
    font-size: 16px;
}

/* Estilo inicial del div, seleccionado por su ID único */
#caja-colorida {
    background-color: rgb(173, 211, 173); /* Color de fondo verde inicial */
    border: 5px solid black;
    padding: 10px;
    margin-top: 20px;
    /* Esto hace que la caja se ajuste al contenido pero en su propia línea */
    width: fit-content; 
}

/* Clase CSS que se añadirá y quitará con JavaScript */
.blue-bg {
    background-color: aqua;
}

JavaScript (script.js): La lógica interactiva.

// Se declara la función que será llamada por el botón
function changeColor() {
    // 1. Seleccionamos el elemento del DOM usando su ID
    const miDiv = document.getElementById('caja-colorida');

    // 2. Usamos el método 'toggle' para alternar la clase 'blue-bg'
    miDiv.classList.toggle('blue-bg');
}

2. Conceptos Clave de CSS

  • Selectores: Son la forma en que "apuntamos" a los elementos HTML.
    • Por etiqueta: body { ... } - Afecta a todas las etiquetas de ese tipo.
    • Por ID: #caja-colorida { ... } - Afecta al ÚNICO elemento con ese id. Se usa para elementos específicos.
    • Por Clase: .blue-bg { ... } - Afecta a TODOS los elementos que tengan esa class. Se usa para estilos reutilizables.
  • El Modelo de Cajas (Box Model): Descubrimos que los elementos como <div> son por defecto display: block. Esto significa que ocupan todo el ancho disponible, como un libro en una estantería, empujando a otros elementos debajo.
  • La Propiedad display: Para controlar este comportamiento, aprendimos sobre:
    • display: block;: Ocupa todo el ancho y se coloca en una nueva línea.
    • display: inline;: Ocupa solo el espacio de su contenido y se coloca al lado de otros, como palabras en una frase.
    • display: inline-block;: Un híbrido. Ocupa solo su espacio (como inline) pero respeta propiedades de caja como padding y border (como block).
  • Ajustar el Ancho: Vimos que width: fit-content; es una solución moderna y directa para hacer que un elemento de bloque ajuste su ancho al contenido que tiene dentro, sin dejar de ocupar su propia línea.

3. Profundizando en JavaScript (La Interacción)

JavaScript es el cerebro que nos permite responder a las acciones del usuario.

  • El DOM (Document Object Model): Es la representación de nuestro HTML como un árbol de objetos. JavaScript puede interactuar con este árbol para modificar cualquier cosa de la página. La puerta de entrada es el objeto document.
  • Seleccionar Elementos: La función document.getElementById('id-del-elemento') es la forma más rápida y segura de "agarrar" un elemento único del DOM para poder manipularlo.
  • Desglose de miDiv.classList.toggle('blue-bg'):
    1. miDiv: Es la variable que contiene el objeto del DOM (nuestro `div`). Es el "quién".
    2. .classList: Es una propiedad de ese objeto. Nos da acceso a su lista de clases. Es el "qué parte".
    3. .toggle(): Es un método (una acción) de classList. Actúa como un interruptor. Es el "qué hacer".
    4. ('blue-bg'): Es el argumento que le pasamos al método. Le dice al interruptor qué clase específica debe alternar. Es el "con qué".
  • Notación de Punto: La estructura objeto.propiedad.metodo() es la forma jerárquica y lógica de acceder a las capacidades de los objetos en JavaScript.
