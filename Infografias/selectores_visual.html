Selectores CSS

Tipos de selectores con ejemplos visuales y código: Tipo (p, h1), Clase (.btn), ID (#header), Atributo (input[type="email"]), Combinadores (nav a, ul > li, h2 + p, h2 ~ p), Pseudo-clases (:hover, :focus, :active, :disabled, :checked, :nth-child(n)) y Pseudo-elementos (::before, ::after, ::placeholder).

1) Selectores de tipo (por etiqueta)

Seleccionan elementos por su nombre de etiqueta HTML. Van al grano, pero son generales (baja especificidad).

Ejemplo A

h1 (título)

párrafo 1

párrafo 2

Ejemplo B

otro h1

otro p

otro p

En este demo, se aplicarían reglas a todos los p o todos los h1.

Código

p { line-height: 1.6; }
h1 { font-size: clamp(1.6rem, 2.8vw, 2.2rem); }

2) Selector de clase (.btn)

Aplica estilos a cualquier elemento con la clase indicada. Es la forma más habitual de tematizar componentes reutilizables.

a.btn span.btn

Código

.btn { padding:.5rem .9rem; border-radius:999px; }
.btn.primary { background: #22c55e; }
a.btn { text-decoration: none; }

3) Selector de ID (#header)

Selecciona un elemento único por su atributo id. Su especificidad es alta: úsalo con moderación en CSS.

#header
main
footer
El selector #header solo afecta a ese elemento único.

Código

#header { background: linear-gradient(#3b82f6, #2563eb); color:#061022; }

4) Selector de atributo (input[type="email"])

Permite apuntar a elementos según atributos y valores. Hay variantes como [attr], [attr^="ini"], [attr$="fin"], [attr*="contiene"].

Estilizamos solo el input[type="email"] con borde y fondo diferenciado.

Código

input[type="email"] { border:2px solid #22c55e; background:#0b2a20; }

5) Combinadores: descendiente, hijo directo, adyacente y hermanos

Relacionan elementos entre sí en el árbol DOM.

Descendiente: nav a

Enlace 1
Enlace 2 (anidado)
Enlace suelto (no coincide)

Hijo directo: ul > li

  • Item 1
  • Item 2
  • Anidado
    • Subitem (no hijo directo)

Adyacente: h2 + p

Título

Párrafo justo después (coincide)

Otro párrafo (no adyacente directo)

Hermanos generales: h2 ~ p

Párrafo antes (no coincide)

Título

Hermano 1 (coincide)

Intermedio

Hermano 2 (coincide)

Código

/* Descendiente (espacio) */ nav a { color: deepskyblue; }
/* Hijo directo (>)     */ ul > li { font-weight: 700; }
/* Adyacente (+)        */ h2 + p { margin-top: 0; }
/* Hermanos (~)         */ h2 ~ p { color: goldenrod; }

6) Pseudo-clases: estado e información estructural

Apuntan a estados (interacción o formulario) o a patrones de posición en el árbol (como :nth-child).

Interacción: :hover, :focus, :active, :disabled

Pasa el ratón (hover), haz tab (focus) o clic (active) para ver cambios.

Formularios / estructura: :checked, :nth-child(n)

En la lista, :nth-child(odd) alterna fondo y :nth-child(3n) resalta múltiplos de 3.

Código

.btn:hover { background:#17315e; }
.btn:focus { outline:2px solid #9ad8ff; outline-offset:2px; }
.btn:active { transform: translateY(1px); }
.btn:disabled { opacity:.5; cursor:not-allowed; }

input:checked + label { color:#22c55e; font-weight:800; }
li:nth-child(odd) { background:#0f294e; }

7) Pseudo-elementos: contenido generado / partes del control

::before y ::after insertan contenido generado; ::placeholder estiliza el texto de placeholder en inputs.

Etiqueta decorada (.tagline)

La estrella es ::before y la flecha es ::after.

El placeholder tiene color/estilo propios con ::placeholder.

Código

.tagline::before { content:"★"; /* decoración al inicio */ }
.tagline::after  { content:"→"; /* decoración al final  */ }
input::placeholder { color:#94a3b8; font-style:italic; }

8) Snippets resumidos

/* Tipo */
p { margin: .5rem 0; } h1 { font-weight: 800; }

/* Clase / ID */
.btn { padding:.5rem .9rem; border-radius:999px; }
#header { position: sticky; top: 0; z-index: 10; }

/* Atributo */
input[type="email"] { border:2px solid #22c55e; }

/* Combinadores */
nav a { text-decoration: none; }
ul > li { list-style: square; }
h2 + p { margin-top: 0; }
h2 ~ p { opacity: .85; }

/* Pseudo-clases y pseudo-elementos */
a:hover { text-decoration: underline; }
input:focus { outline: 2px solid #9ad8ff; outline-offset: 2px; }
li:nth-child(odd) { background: #0f294e; }
.badge::before { content: "●"; margin-right:.4rem; }

Tip: Combina selectores con buen criterio y evita subir la especificidad innecesariamente. Prioriza clases sobre IDs para mantener CSS escalable (BEM, utilidades, etc.).
