Herramientas vs Taller

Una librería es una herramienta concreta (como un taladro): la usas cuando quieres y como quieres. Un framework es un taller completo: tiene mesas, normas y flujo; trabajas dentro de su organización.

Definiciones

¿Qué es cada cosa?

  • Librería: conjunto de funciones/clases que tú llamas cuando lo necesitas.
  • Framework: esqueleto que define estructura y ciclo de vida; te llama a ti (Inversión de Control).
Idea clave: una librería te da piezas; un framework te da andamiaje y reglas.

Esquema

Control del flujo: ¿quién llama a quién?

Librería: tu programa controla todo y usa la librería como apoyo.

Framework: el framework posee el ciclo de vida y “te enchufas” en sus puntos de extensión.

Alcance típico

Las librerías suelen cubrir una parte; los frameworks abarcan varias capas.

Matices

Micro‑frameworks

Flask o Express opinan poco y se sienten como varias librerías coordinadas. Útiles para empezar rápido sin perder flexibilidad.

# Flask (Python)
from flask import Flask
app = Flask(__name__)

@app.get("/")
def home():
    return "Hola, web"

app.run()
Matices

¿React es framework?

React es una librería de UI, pero con Next.js/Remix y tooling alrededor, la experiencia es “tipo framework”.

// React (JS)
function Button({ onClick, children }){
  return <button onClick={onClick}>{children}</button>
}
// Tú controlas cuándo y cómo usarlo
Elección

Árbol de decisión (simplificado)

Checklist rápido
  • Proyecto pequeño o una función clara → librerías.
  • Producto completo, equipos y estándares → framework.
  • Necesitas velocidad sin ataduras → micro‑framework.
  • Maximizar libertad arquitectónica → librerías.
Glosario

Tres conceptos para recordar

Inversión de Control (IoC): el framework controla el flujo y llama a tu código en puntos de extensión.
Principio de Hollywood: “Don’t call us, we’ll call you”.
Tooling: herramientas integradas (CLI, dev server, scaffolding, pruebas, build) que aceleran el trabajo en frameworks.

Consejo: empieza pequeño y evoluciona hacia frameworks cuando el producto y el equipo lo pidan.
