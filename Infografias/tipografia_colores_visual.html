1) Tipografía y texto

Base de legibilidad: familia, tamaño, peso, estilo y composición (interlineado, tracking, alineación, transformación y decoración).

Familias (font-family) y pila de fallbacks

Sistema Interfaz moderna
Serif Lectura clásica
Monospace Código & datos

Especifica varias fuentes en orden: el navegador usa la primera disponible. Ej.: font-family: Inter, system-ui, -apple-system, Segoe UI, Roboto, sans-serif;

Tamaño (font-size)

.75rem .875rem 1rem 1.125rem 1.25rem clamp(1.6rem, 2.8vw, 2.2rem)

Peso y estilo (font-weight, font-style)

300 (Light) 400 (Normal) 600 (SemiBold) 700 (Bold) Italic

Interlineado (line-height)

line-height: 1.1 — más compacto
line-height: 1.5 — legible para párrafos
line-height: 1.9 — aireado

Tracking (letter-spacing)

Letter Letter Letter

Decoración (text-decoration)

Código

:root { --font: Inter, system-ui, -apple-system, Segoe UI, Roboto, sans-serif; }
body { font: 400 16px/1.55 var(--font); letter-spacing: .2px; }
h1 { font-weight: 700; text-transform: uppercase; }
p.note { line-height: 1.6; }
.title-tight { letter-spacing: .04em; }
.center { text-align: center; }
.badge { text-transform: capitalize; text-decoration: underline; }

2) Colores: #hex, rgb(), hsl(), transparent

Cuatro formas comunes de definir color. transparent deja pasar el fondo. HSL es muy útil para crear paletas.

#ef4444
rgb(34, 197, 94)
hsl(220 80% 52%)
transparent

Código

h1 { color: #0ea5e9; }
.ok { color: rgb(22,163,74); }
.brand { color: hsl(220 90% 55% / 1); } /* admite alfa: hsl(h s l / a) */
.muted { color: transparent; }

3) Fondos: background*

Capas de fondo: color, imágenes, gradientes. Controla tamaño, posición y repetición.

Repetición (tile)

Patrón generado con gradientes + repeating-linear-gradient.

Imagen: cover

background-size: cover recorta para cubrir; centrado con background-position: center.

Imagen: contain

contain ajusta manteniendo proporción sin recorte; puedes fijar la posición.

Posición

Código

.hero {
  background:
    linear-gradient(#0005, #0003),
    url(header.jpg);
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
}

4) Sombras y filtros: box-shadow, text-shadow, filter

Profundidad y efectos visuales. box-shadow admite múltiples capas; text-shadow resalta títulos; filter transforma una imagen ya renderizada.

box-shadow
text-shadow

filter: grayscale(100%)

filter: blur(2px)

filter: contrast(130%) brightness(110%)

Consejo: evita filtros pesados en móviles o anima con cuidado.

Código

.card { box-shadow: 0 22px 44px -18px rgba(0,0,0,.45), 0 10px 24px -12px rgba(59,130,246,.35); }
h1 { text-shadow: 0 2px 10px rgba(0,0,0,.3); }
img.thumb { filter: grayscale(100%) contrast(110%); }

5) Snippets resumidos

/* Tipografía base */
:root { --font: Inter, system-ui, -apple-system, Segoe UI, Roboto, sans-serif; }
body  { font: 400 16px/1.6 var(--font); letter-spacing:.2px; }

/* Colores */
.title { color: hsl(220 90% 55%); }
.muted { color: rgb(100 116 139); }

/* Fondos */
.hero {
  background: linear-gradient(#0005, #0003), url(header.jpg);
  background-size: cover; background-position: center; background-repeat: no-repeat;
}

/* Sombras/Filtros */
.card { box-shadow: 0 10px 30px -12px rgba(0,0,0,.45); }
h1   { text-shadow: 0 2px 8px rgba(0,0,0,.3); }
img  { filter: grayscale(100%); }

Tip: usa escalas de font-size y line-height consistentes, y construye tu paleta con HSL para variar tono/saturación/luz de forma sistemática.
